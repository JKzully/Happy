# HCE Dashboard
Dashboard webapp fyrir Happy Hydrate - íslenskt vörufyrirtæki sem selur drykki í verslunarkeðjum.

## Tech Stack
- Next.js 14 (App Router) + TypeScript
- Supabase (PostgreSQL + Auth)
- Tailwind CSS + Shadcn/ui
- Recharts fyrir gröf
- Vercel hosting

## Tungumál
Allt UI á íslensku. Breytunöfn og kóði á ensku.

## Keðjur
Krónan (27 útibú), Samkaup (Nettó/Kjörbúðir/Iceland/Extra), Bónus (34), Hagkaup (7), Shopify, N1

## Vörur
Hydration Xpress (5 bragðtegundir), Creatine Xpress (2), Energy Xpress (1), Kids (2)

## Mikilvægt
- Dark/light theme toggle
- Excel upload fyrir söluskýrslur frá keðjum
- Verð breytilegt eftir keðju
- Minimal/clean útlit, mikið whitespace
